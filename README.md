# Node Note / 节点随记

A powerful Blender add-on for adding text and image annotations directly to nodes.

[中文说明](#中文说明)

---

## Text Notes

Add customizable text annotations to any node with options for:
- Font size and colors
- Background styling
- Alignment and positioning
- Offset adjustments

## Image Notes

Attach reference images directly to nodes:
- Paste images from clipboard
- Adjustable image width
- Flexible positioning
- Pack/unpack support for portability

## Index/Badge

Numbered badges for node identification:
- Interactive numbering mode - click to assign
- Customizable badge colors
- Easy navigation between numbered nodes

## 中文说明

Blender 强大的节点图文标注工具。

### 文本笔记

为任意节点添加可自定义的文字注释：
- 字体大小和颜色
- 背景样式
- 对齐方式
- 位置偏移调整

### 图像笔记

直接在节点上附加参考图片：
- 从剪贴板粘贴图片
- 可调节的图片宽度
- 灵活的定位方式
- 打包/解包支持

### 序号/徽章

用于节点识别的编号标记：
- 交互编号模式 - 点击分配序号
- 自定义徽章颜色
- 快速导航到已编号节点
