# Node Note / 节点随记

A powerful Blender add-on for adding text and image annotations directly to nodes.

[中文说明](#中文说明)

---

## Text Notes

Add customizable text annotations to any node with options for:
- Font size and colors
- Background styling
- Alignment and positioning
- Offset adjustments

## Image Notes

Attach reference images directly to nodes:
- Paste images from clipboard
- Adjustable image width
- Flexible positioning
- Pack/unpack support for portability

## Index/Badge

Numbered badges for node identification:
- Interactive numbering mode - click to assign
- Customizable badge colors
- Easy navigation between numbered nodes

## Known Issues/To-Do

- Node timeing/named attribute usage of group  overlap with nodes
- Make drawn text/images not overlap with nodes
- Saving v5.0+ files in v4.5 results in missing notes when reopening in v5.0+.

# 中文说明

Blender 强大的节点图文标注工具。

## 文本笔记

为任意节点添加可自定义的文字注释：
- 字体大小和颜色
- 背景样式
- 对齐方式
- 位置偏移调整

## 图像笔记

直接在节点上附加参考图片：
- 从剪贴板粘贴图片
- 可调节的图片宽度
- 灵活的定位方式
- 打包/解包支持

## 序号/徽章

用于节点识别的编号标记：
- 交互编号模式 - 点击分配序号
- 自定义徽章颜色
- 快速导航到已编号节点

## 已知问题/待办

- 节点耗时/命名属性使用 和 笔记重叠
- 让绘制的文本/图像和文本重叠的地方不绘制
- 5.0以上版本的文件,用4.5打开保存后,5.0再打开笔记会丢失.
